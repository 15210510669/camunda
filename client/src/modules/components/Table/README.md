Example

```js
const {THead, TBody, TH, TR, TD} = Table;

<Table>
  <THead>
    <TR>
      <TH>header 1</TH>
      <TH>header 2</TH>
    </TR>
  </THead>
  <TBody>
    <TR>
      <TD>cell 1 a</TD>
      <TD>cell 2 a</TD>
    </TR>
    <TR>
      <TD>cell 1 b</TD>
      <TD>cell 2 b</TD>
    </TR>
  </TBody>
</Table>
```
